📝 ブログ記事の追加

  1. 「ブログ管理」を開く
  2. 「新規作成」タブへ
  3. 必要事項を入力
  4. 「コードを生成」をクリック
  5. コードをコピー
  6. /data/blogs.js を開く
  7. 先頭の [ の直後に貼り付け
  8. 保存してブラウザで確認

📋 事例の追加

  1. 「事例管理」を開く
  2. 「新規追加」タブへ
  3. 事例情報を入力
  4. 「コードを生成」をクリック
  5. /data/cases.js に追加

⚙️ 共通設定の変更

  1. /data/config.js を開く
  2. LINEのURLを更新
  3. 会社情報を更新
  4. 保存で全ページに反映
